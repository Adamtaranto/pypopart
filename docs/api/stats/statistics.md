# Network Statistics

Network topology and graph statistics.

## Overview

The `statistics` module calculates network-level metrics:

- Basic properties (nodes, edges, density)
- Centrality measures
- Path metrics (diameter, average path length)
- Clustering coefficients
- Community detection
