# File Writers

Network export functionality for various formats.

## Overview

Export networks in multiple formats:

- **NEXUS**: Compatible with PopART
- **GML**: Graph Modeling Language
- **GraphML**: XML-based graph format
- **JSON**: JavaScript Object Notation
